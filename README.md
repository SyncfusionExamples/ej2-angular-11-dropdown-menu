# Angular11DropdownMenu
The [Angular Dropdown Menu](https://www.syncfusion.com/angular-ui-components/angular-dropdown-menu?utm_source=github&utm_medium=listing&utm_campaign=angular-dropdown-menu-github-samples) is a graphical user interface component that lets users choose from a list of actions that can be triggered. ## Dropdown Menu position

Customize position values to display the Angular Dropdown Menu above, to the right, or to the left of the dropdown button.

The popup menu is a toggleable container that holds a list of action items or custom content that will open or close when the button is clicked. All the items in a dropdown menu can be customized.
